# Japanese Verb Conjugation

A python application for conjugating Japanese verbs.

## Instalation

This is a collection of python scripts. If you have python installed on your machine, you should be able to download the files and run it. The only third-party package used in this application is **Pandas**.

## Use

The main use of this application is to conjugate Japanese verbs. This application only covers the approximatly 6 - 9 common conjugation forms found at the N5 level. For higher levels please seek out more advanced resources.

To actually run the application, the following console command can be executed:

```
> python main.py
```
